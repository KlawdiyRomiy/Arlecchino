Improved
- Arlecchino remains on the beta release track with focused frontend stability and interaction polish.
- Floating panel drag tracking and panel motion behavior are smoother after removing unnecessary frontend transition delays.
- Close confirmation dialogs now animate more consistently with the rest of the desktop UI.
- Agent instruction docs were streamlined for clearer local development guidance.

Fixed
- Diagnostics underlines and git gutter readiness are more reliable in CodeMirror editor surfaces.
- React Doctor frontend errors from the recent UI cleanup pass have been fixed.
- AI chat activity status rendering was split into a smaller helper module without changing the beta release channel.

Compatibility
- Existing alpha updater clients can still move to this beta through the legacy alpha-channel manifest asset in the same GitHub release.
- Wails runtime remains pinned to `v3.0.0-alpha.95`; this is upstream dependency status, not Arlecchino product status.

Security
- macOS beta artifacts remain unsigned/not notarized unless Developer ID signing and notarization are explicitly configured.
- Update manifests still require SHA256 and Ed25519 verification before an update can be staged.
