Improved
- Indexing progress updates now fan out less aggressively across the shell and problem surfaces.
- Access-member autocomplete can use proof-backed auto-imports from dependency catalogs and virtual LSP import context.
- CodeMirror completion edit application is safer around active completion sessions and ghost text.

Fixed
- LSP completion fallback resolution is more resilient when completion details arrive through fallback paths.
- Window controls bubble alignment is corrected.

Compatibility
- Arlecchino advances to the `0.2.6-beta` version line for this build.
- Existing alpha updater clients can still move to this beta through the legacy alpha-channel manifest asset in the same GitHub release.
- Wails runtime remains pinned to `v3.0.0-alpha.98`; this is upstream dependency status, not Arlecchino product status.

Security
- macOS beta artifacts use local certificate identity stability for owner/tester updates when configured, but they are still not Apple Developer ID signed or notarized.
- Update manifests still require SHA256 and Ed25519 verification before an update can be staged.
