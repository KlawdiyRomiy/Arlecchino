Improved
- Panel slot ownership now stays more predictable when side panels are reopened, moved, or restored through shortcuts.
- The reopen shortcut now routes through the panel layout model more consistently.
- macOS window controls now stay steadier during window dragging and native control state transitions.

Fixed
- Definition chooser positioning is more stable around editor selection and quick-look placement.
- Panel event routing now preserves ownership state more reliably across layout changes.
- Native macOS window control updates now avoid stale visibility and drag-state transitions.

Compatibility
- Arlecchino advances to the `0.2.15-beta` version line for this build.
- Existing alpha updater clients can still move to this beta through the legacy alpha-channel manifest asset in the same GitHub release.
- Wails runtime remains on `v3.0.0-alpha2.103`; this is upstream dependency status, not Arlecchino product status.

Security
- macOS beta artifacts use local certificate identity stability for owner/tester updates when configured, but they are still not Apple Developer ID signed or notarized.
- Update manifests still require SHA256 and Ed25519 verification before an update can be staged.
