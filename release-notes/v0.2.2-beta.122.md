Improved
- Renderer font-size scaling now avoids broad shell rerender pressure, improving panel and editor responsiveness during UI updates.
- Dependency sync now discovers nested dependency manifests instead of only treating the workspace root as the dependency surface.

Fixed
- Wails runtime is rolled back to `v3.0.0-alpha.95` after `v3.0.0-alpha.96` showed noticeably worse animation smoothness in the IDE shell.
- macOS local certificate code identity detection now reads verbose codesign authority data before classifying update candidates.
- Dependency sync execution tests now track the stable action IDs used by the current nested manifest planner.

Compatibility
- Existing alpha updater clients can still move to this beta through the legacy alpha-channel manifest asset in the same GitHub release.
- Wails runtime remains pinned to `v3.0.0-alpha.95`; this is upstream dependency status, not Arlecchino product status.

Security
- macOS beta artifacts use local certificate identity stability for owner/tester updates when configured, but they are still not Apple Developer ID signed or notarized.
- Update manifests still require SHA256 and Ed25519 verification before an update can be staged.
