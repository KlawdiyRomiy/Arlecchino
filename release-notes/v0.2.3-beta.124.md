Improved
- Startup now keeps less frontend and backend state resident before the IDE needs it.
- Dependency sync preflight and policy surfaces now expose clearer runtime data before execution.

Fixed
- Autocomplete member-access filtering now keeps useful LSP-proven completions while dropping mismatched library namespace noise.
- Completion metadata merging no longer carries import/text-edit side effects across unrelated autocomplete sources.

Compatibility
- Arlecchino advances to the `0.2.3-beta` version line for this build.
- Existing alpha updater clients can still move to this beta through the legacy alpha-channel manifest asset in the same GitHub release.
- Wails runtime remains pinned to `v3.0.0-alpha.95`; this is upstream dependency status, not Arlecchino product status.

Security
- macOS beta artifacts use local certificate identity stability for owner/tester updates when configured, but they are still not Apple Developer ID signed or notarized.
- Update manifests still require SHA256 and Ed25519 verification before an update can be staged.
