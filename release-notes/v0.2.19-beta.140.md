Improved
- The welcome workspace screen has a more focused layout for opening, creating, cloning, and returning to recent projects.
- Theme selection is now available directly from the welcome screen, including custom theme import.
- Workspace zoom shortcuts now work on the welcome screen before a project window is opened.

Changed
- Wails runtime is updated to `v3.0.0-alpha2.113`.

Compatibility
- Arlecchino advances to the `0.2.19-beta` version line for this build.
- Existing alpha updater clients can still move to this beta through the legacy alpha-channel manifest asset in the same GitHub release.
- Wails runtime is updated to `v3.0.0-alpha2.113`; this is upstream dependency status, not Arlecchino product status.

Security
- Private GitHub release manifests and update artifacts continue to use authenticated release asset access while the repository remains private.
- macOS beta artifacts use local certificate identity stability for owner/tester updates when configured, but they are still not Apple Developer ID signed or notarized.
- Update manifests still require SHA256 and Ed25519 verification before an update can be staged.
