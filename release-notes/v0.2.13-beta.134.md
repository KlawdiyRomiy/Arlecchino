Improved
- AI chat, Git review, and Problems state now stay tied to the active project more consistently, reducing cross-workspace state bleed.
- Manual diagnostics full scans are more resilient when language servers or project state change during a scan.
- Interactive surfaces, floating panels, and inline agent controls now keep steadier motion during active IDE work.

Fixed
- Git and Problems indicators now follow the selected project more reliably.
- AI chat sessions are now scoped by project root in the backend runtime path as well as the frontend state path.

Compatibility
- Arlecchino advances to the `0.2.13-beta` version line for this build.
- Existing alpha updater clients can still move to this beta through the legacy alpha-channel manifest asset in the same GitHub release.
- Wails runtime updates to `v3.0.0-alpha2.103`; this is upstream dependency status, not Arlecchino product status.

Security
- macOS beta artifacts use local certificate identity stability for owner/tester updates when configured, but they are still not Apple Developer ID signed or notarized.
- Update manifests still require SHA256 and Ed25519 verification before an update can be staged.
