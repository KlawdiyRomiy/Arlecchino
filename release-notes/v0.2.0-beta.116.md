Improved
- Arlecchino remains on the beta release track with focused editor, AI chat, and macOS polish.
- Editor tabs now expose a direct Find in File control for the active CodeMirror editor.
- AI Chat fullscreen mode can route its common fullscreen shortcuts to history, session search, and review surfaces without opening the global panels behind it.
- Editor fold gutter geometry is more stable under adaptive performance pressure.

Fixed
- macOS app icon override sizing now preserves the app icon body more carefully across generated representations.
- The native macOS bridge and shell menu shortcut metadata now stay aligned with the fullscreen AI Chat shortcut routing.

Compatibility
- Existing alpha updater clients can still move to this beta through the legacy alpha-channel manifest asset in the same GitHub release.
- Wails runtime remains pinned to `v3.0.0-alpha.95`; this is upstream dependency status, not Arlecchino product status.

Security
- macOS beta artifacts remain unsigned/not notarized unless Developer ID signing and notarization are explicitly configured.
- Update manifests still require SHA256 and Ed25519 verification before an update can be staged.
